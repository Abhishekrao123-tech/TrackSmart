# 🚀 TrackSmart

TrackSmart is a full-stack expense tracking web application that helps users manage their income and expenses efficiently. Users can securely sign up, log in, add transactions, track spending, and visualize financial data through interactive charts.

## ✨ Features

* 🔐 JWT Authentication (Signup/Login)
* 💰 Add & Delete Transactions
* 📊 Monthly Spending Analytics
* 🥧 Category-wise Expense Charts
* 👤 User Profile Management
* 🔒 Protected Routes
* 🗄️ MySQL Database Integration
* ☁️ Fully Deployed Application

## 🛠 Tech Stack

**Frontend**
